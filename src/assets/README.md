# Portfolio Assets

## Images

Place your portfolio images in this directory:

- `profile.jpg` - Your professional headshot for the hero and about sections
- `projects/` - Screenshots or preview images of your projects
- `icons/` - Custom icons if needed

## Recommended Image Sizes

### Profile Photo
- **Hero section**: 400x400px (square, high quality)
- **About section**: 300x300px (square)

### Project Screenshots
- **Thumbnail**: 400x300px (4:3 aspect ratio)
- **Featured image**: 800x600px (4:3 aspect ratio)

## Image Optimization Tips

1. Use WebP format for better compression
2. Provide fallback formats (JPEG/PNG) for older browsers
3. Compress images to reduce file size while maintaining quality
4. Consider using a CDN for better performance
